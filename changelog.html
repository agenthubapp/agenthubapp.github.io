Changelog

What's new in AgentHub

A running log of releases, features and fixes. Newest first.

v3.0 — Native macOS app Latest Jun 2026

AgentHub leaves the terminal.

  • Ships as a signed & notarized .dmg — install it like any Mac app, no command line required.
  • Self-running app shell (Electron) bundling the web app + Jira worker; your data lives in Application Support.
  • Open on phone — a QR code and a stable Tailscale URL, with a Doctor connectivity check.
  • Login-shell PATH resolution so the Finder-launched app finds your CLIs (your agent, gh, tuist…).
Hardening & Metrics Jun 7, 2026

Security, durability and visibility.

  • Optional shared-token auth on every page and API; Jira tokens encrypted at-rest (AES-256-GCM).
  • SQLite WAL + busy-timeout, Prisma Migrate, and a one-command VACUUM-INTO backup.
  • Metrics dashboard — outcomes, success / verify-green rates, diff-coverage, PRs, run time and token cost.
  • Vitest suites for the gate / state-machine / crypto logic, wired into GitHub Actions CI.
Deep QA, Knowledge & Autopilot Jun 6, 2026

The quality engine and the brain.

  • Deep QA engine — diff-coverage, quality gates + a bounded auto-fix loop, cross-model AI review and a plan-checker.
  • Security & dead-code scans (gitleaks, semgrep, periphery) plus a Quality Scorecard on every PR.
  • Project Wiki with local semantic search (Ollama bge-m3), Confluence import and attachments → context (videos → frames).
  • Ask — project Q&A grounded in your wiki, tickets and runs.
  • Autopilot — batch runs with a conflict-aware scheduler → a Draft PR per ticket, plus a realtime agent log.
Agent execution & worktrees Jun 4, 2026

Agents that edit real branches.

  • Full run lifecycle + a plan-review gate (approve, or request changes → it replans).
  • Per-ticket git worktrees (branch = the ticket key off develop); up to 3 runs in parallel.
  • Local Changes card — review the diff, commit (conventional) and open a PR when you're happy.
  • Works with whatever agent CLI you run — not locked to any one vendor.
Context Pack Jun 4, 2026

Give the agent the full picture.

  • Per-ticket Context Pack aggregating Jira, Confluence, Figma links and related repo history.
  • Token-budgeted, with per-source status and one-click copy.
Orchestration shell Jun 4, 2026

The mission-control foundation.

  • Live Jira board (six columns) synced by a standalone polling worker.
  • Ticket workspace, settings, an "only my tickets" filter, and SQLite persistence — zero infra.
