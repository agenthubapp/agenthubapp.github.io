Changelog

What's new in AgentHub

A running log of releases, features and fixes. Newest first.

v0.8.6 — Design in Figma + smarter free models Latest Jul 5, 2026

Design → Figma.

  • Design → Figma — New Design page: describe a screen in plain language and an agent draws it directly into Figma (frames, text, components) via a write-capable Figma MCP, with a live preview and design-token grounding.
  • Smarter free / auto models — When a task runs on a free or small model, AgentHub now gates its output with a stronger, independent reviewer (cross-tier review), forces a plan check, tightens scope enforcement, and adds an extra auto-fix pass — without weakening any quality gate.
v0.8.5 — mobile polish + snappier runs Jul 4, 2026

Instant run feedback.

  • Instant run feedback — starting a run from a ticket now shows a loading toast, a 'Starting…' button, and the run cockpit appears immediately (no more wondering whether it's working).
  • Revamped iOS onboarding — the pairing screen got a premium dark redesign with the real app icon and a readable title (it was previously invisible on a light background).
  • Live Activity fixes — the Lock Screen / Dynamic Island run tracker now renders its full card (phase, status, progress) on iOS 26, and the expanded island shows the full ticket key.
v0.8.4 — Live Activity fix Jul 4, 2026

Fixed the iOS Live Activity.

  • Fixed the iOS Live Activity — the revamped Lock Screen / Dynamic Island run tracker introduced in v0.8.3 rendered only the header on iOS 26 (the phase chip, status line, and progress bar were blank). The layout is rewritten and now shows the full card again.
v0.8.3 — clearer plans, sharper UI Jul 4, 2026

Plan at a glance.

  • Plan at a glance — the plan-approval screen now shows a Needs / Does / Impact summary: blocking questions up top, the predicted blast radius (which files a change touches), and an acceptance checklist — with the raw plan one click away.
  • UI hygiene checks — every UI change is scored for design debt (missing empty/error states, generic copy, type-scale sprawl, off-grid spacing) even when there is no Figma link, and the agent is guided to hold that bar while it builds.
  • Live Activity redesign — the iOS Lock Screen and Dynamic Island run tracker got a cleaner look: icon badge, phase pill, and a crisp progress bar.
  • Polish & fixes — light-mode legibility fixes across the app, a command-palette keyboard-selection bug, and mobile stability fixes (reconnect no longer resets the screen, Ask no longer hangs on errors).
v0.8.2 — Verify tab fixes Jul 4, 2026

Verify on device now reachable for every ticket.

  • Verify on device now reachable for every ticket — the simulator review tab was only appearing for UI tickets with screenshots; it now shows on any ticket that has run work, including backend tickets in Ready for Review.
  • Commit-message suggestions — more reliable, and when the AI model can't be reached it now tells you the real reason instead of a generic 'could not parse' message.
v0.8.1 — Verify on device, native mobile upgrade & hardened relay Jul 4, 2026

Verify on device.

  • Verify on device — one tap installs the ticket's build on the simulator, replays its recorded flow to the exact screen, and streams a live view alongside the ticket's acceptance checklist.
  • Native mobile app leveled up — new More tab with Autopilot control, Settings and a real terminal; ticket verify view with live simulator snapshots; Live Activities + Dynamic Island for active runs; smarter Lock Screen widgets.
  • Relay security hardening — device pairing now runs fully end-to-end encrypted through the relay, plus registration tokens, rate limits and abuse protection for public relays.
  • Mobile web polish — the iOS keyboard no longer covers the chat/terminal input, bigger touch targets everywhere, PWA icon and theme fixes.
  • Cloudflare Tunnel option — free self-hosted remote access with a step-by-step guide (shown in Settings only once configured).
  • Housekeeping — locally created tickets can now be deleted with full cleanup.
v0.8.0 — beyond Jira, beyond mobile Jul 3, 2026

GitHub Issues tracker.

  • GitHub Issues tracker — point a project at any `owner/repo` and AgentHub syncs its issues as tickets, using your existing `gh` CLI login. No token to paste, nothing stored. Issue labels like "in progress" and "review" land in the right board columns automatically.
  • Local-only projects — no tracker at all: create tickets right in AgentHub with the new **New ticket** button (keys like `MYAPP-1`), run agents on them, and push them to Jira/GitHub later if you want.
  • Web projects (Node/npm) — a new `Web · Node/npm` build stack: `npm run build/test/lint` verification, diff-coverage from lcov, and UI proof captured with **Playwright** screenshots — the same Figma fidelity judge and quality scorecard the mobile stacks get.
  • Jira is now an adapter, not a requirement — the whole pipeline (sync, status transitions, context packs, attachments, filing review findings) runs through a tracker interface, so Jira, GitHub Issues, and local-only projects all behave consistently. Also fixes multi-project installs where opening a PR transitioned Jira with the wrong credentials.
  • Doctor gains a **Web build & test** section; project setup lets you pick Jira / GitHub / None per project.
v0.7.3 — Edit projects & fixed setup dialog Jul 3, 2026

Edit project.

  • Edit project — a pencil button in the project switcher (and the "Jira not configured — set up" link in the header) opens the project for editing: name, Jira credentials, repo, base branch and verify config. No more re-creating a project just to add Jira.
  • Saved token stays safe — leaving the API-token field blank keeps the stored token; "Test connection" works against the saved token too.
  • Setup dialog fixed — the Add/Edit project dialog no longer renders clipped under the glass header; it is properly centered on every page.
  • Delete any project — the last remaining project can now be removed too; AgentHub returns to first-run setup.
v0.7.2 — Honest agent errors, custom gateways & debug bundle Jul 3, 2026

No more garbage answers in chat.

  • No more garbage answers in chat — when an agent CLI fails (out of credit, expired login, stray terminal codes), Ask now fails over to the next agent and reports the real cause with a fix hint, instead of posting the raw error as a reply.
  • Actionable auth & billing errors — "Login canceled", "Credit balance is too low" and similar failures now pause runs with the exact next step (re-login command or where to fix the key), in both the run panel and chat.
  • Codex behind a company AI gateway now works — if your Codex CLI goes through a LiteLLM / corporate gateway, paste that gateway's token in Settings → AI Connection and AgentHub uses it correctly (it no longer rejects gateway tokens as invalid OpenAI keys).
  • Debug bundle — one click in Settings → System check downloads a diagnostics file (agent status, recent failed-run logs — secrets removed) that you can attach to a bug report instead of screenshots.
  • Desktop app: advanced settings file — power-user options that don't have a UI yet can be set in a plain text file the app creates for you at ~/Library/Application Support/AgentHub/env; error messages and Settings now point to this exact file.
v0.7.1 — Ask 2.0 + platform hardening Jul 3, 2026

Ask 2.0.

  • Ask 2.0 — real-time streaming answers, rich markdown with highlighted code + clickable file references, @file/@folder context mentions with visible context pills, and reviewable diffs with one-click revert in Agent mode
  • Run-in-chat — type "fix ticket KEY" in Ask and the whole run (phases, questions, diff, summary) happens live inside the conversation
  • Inspect redesigned — one-step review cockpit with background sim capture, pins in overlay/compare modes, and mobile support
  • Bring your own AI endpoint — the local "brain" (retrieval rerank, briefings) now speaks OpenAI-compatible; default stays local Ollama, or point it at LM Studio/OpenRouter/any gateway in Settings → AI models
  • Light mode — light/dark/auto theme, flash-free, desktop chrome included
  • Sharper agents — Spec-Contract scope enforcement on every diff, clearer "needs your input" prompts with one-tap options, smarter wiki (learning dedup + health panel)
  • Fixes — pop-out panel no longer breaks at small heights; slimmer desktop app (no stray mobile shell)
v0.7.0 — Inspect: Figma vs sim review Jun 27, 2026

Inspect cockpit.

  • Inspect cockpit — Side-by-side Figma design vs live simulator with split, overlay, and slider compare; zoom, checklist, sign-off, or send pins to the agent for rework
  • Review queue — Ready Review tickets surface on Inspect home; resume draft sessions without slow reloads

  • Settings refresh — Reorganized System, Agents, and Project settings with Figma connection test
