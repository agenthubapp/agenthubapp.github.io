Quality engine

A QA gate that doesn't wave code through.

Every run — manual or autopilot — exits through the same gate. Nothing gets a PR because the diff "looked fine".

  • Diff-coverage, not just "tests pass" — new lines must be tested
  • Cross-model AI review verifies every acceptance criterion
  • Secrets, semgrep & dead-code scans on the changed files
  • Bounded auto-fix: failures go back to the agent with exact errors
  • Baseline-aware: pre-existing failures never block your work

Autopilot

Hand it a backlog. Get back a stack of PRs.

The scheduler builds a file-overlap conflict graph, releases safe tickets in parallel waves, and never auto-merges — you stay the gate.

  • Conflict-aware waves — collisions run sequential, the rest parallel
  • Triage chips flag which tickets are auto-safe before you queue them
  • Token budgets pause a batch before it burns your quota
  • needs-input surfacing — a stuck item asks one question, inline

Nova

An assistant that acts, not just answers.

Nova watches your board around the clock and speaks up with one-tap actions — morning standups, stale-PR reminders, flaky-test fixes, and ready-ticket dispatches. Even from your lock screen.

  • Evidence-based: she picks the agent with the best recent track record
  • Conflict-aware: tickets that collide with in-flight work are held back
  • Never annoying: quiet hours, never repeats herself, backs off when you decline
  • Fully auditable: everything she does lands in one reviewable thread

Local-first by design

Your tickets, code and keys never leave your Mac.

No cloud backend, no telemetry, no per-seat SaaS. One process, one SQLite file, your own agent subscriptions.

Runs entirely on your Mac

SQLite plus a polling worker. Nothing to provision, nothing to pay for.

Agents never push

They edit isolated worktrees. Only you commit, push and merge.

Secrets encrypted at rest

Tokens sealed with AES-256-GCM; the key never leaves the machine.

Even the AI brain is local

Retrieval, reranking and Nova's voice run on Ollama models you host.
